# Weather-App
A responsive Weather Application built with React and Vite that provides real-time weather updates through API integration. Designed with a clean, modern UI and optimized for a seamless user experience across devices.

##  Features

-  Search weather by city name
-  Real-time temperature, humidity, and conditions
-  Fully responsive design for mobile and desktop
-  Clean UI built with Material UI components
-  Fast performance powered by Vite

## Tech Stack

- **React** – Frontend library
- **Vite** – Build tool & dev server
- **Material UI** – UI components
- **Axios** – API integration
  
npm install
npm run dev
